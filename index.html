What are classes and ids?

Selecting by tag like we just did works well when we are making changes to every instance of a tag (all 'h1', all 'p' tags, etc..).

Sometimes, we want only some 'p' tags to look a certain way, or even one specific tag. In order to change specific elements, we use CSS classes and ids.

To implement a class, we will need to write a CSS block just like we did with 'body'. For example: 

    .myClass {
      background-color: gray;
      color: blue
    }

To write declarations for a class, first decide on a name. Writing '.', then the name will let CSS know these are class declarations

To use this class, we just put it inside any HTML element we want, like so:

Now, just this one 'h3' element will have a gray background and blue text! Not only that, any element that is given the class "myClass" will have a gray background and blue text.

Let's practice this. First, create a class in 'style.css'. It can be called anything you like. Your name, your favorite color, etc..

Next, add some declarations. If you're not sure what to add, try some or all of the following, and feel free to change the values:

    .anExampleClass {
      font-size: 300%;
      background-color: lightgreen;
      font-family: courier;
      border: 1px dotted red;
      text-transform: uppercase;
      letter-spacing: 5px;
      transition: rotate(-30)deg;
    }

Finally, with our class declarations created, we need to assign this class to elements in index.html

Open index.html in your IDE, choose some HTML elements and add 'class=' followed by the name of your new class, in quotes. Maybe try adding a class to the `h1` and `h2` elements at the top of the page.

Save the work you've done, and refresh this page. The elements you've selected should display the styling you've declared.
